# NewsAPI
Simple News App using newsapi.org API.

Technologies used:
  RecyclerView
  Webview (https://github.com/TheFinestArtist/FinestWebView-Android)
  Picasso (https://github.com/square/picasso)
  Volley (https://github.com/google/volley)
